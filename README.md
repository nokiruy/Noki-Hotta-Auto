## 功能列表

- **主要功能**
  - 日常活动副本领活跃一条龙（支持UI可视化编轴）
  - 自动宝珠迷宫（支持双路线选择：45-50 / 46-38）
  - 钓鱼赚正义币
  - 角色全装备开孔洗孔（支持多词条筛选）
  - 自动炼金(支持筛选品质)
  - 钟楼无限重开
  - 高分挑战无限重开
  - 技术模拟战自动重开
  - 异时层按轴刷(自动重开直到胜利)
  - 3-9刷技能等级
  - 战斗结束后检测经验未满人数，如果少于设定的值则提醒并暂停脚本

- **辅助功能**
  - 定时启动
  - 电脑关机
  - 启动或结束后打开文件，应用或者网站（网页活动）
  - 启动或结束后弹窗提醒自定义的文本(如提醒你签到微信小助手)

   - 选择最新版本的 `Noki_HBR_Auto-v*.*.*.exe`

- [ADB](https://developer.android.com/tools/adb) 设备控制协议
